## NotebookLM Enhancer (中文)

NotebookLM Enhancer 是一个非官方的浏览器扩展，专为优化 Google NotebookLM 的使用体验而设计。

### 🌟 主要功能

- **结构化目录系统**：提供文件夹管理，支持创建、重命名、拖拽移动。
- **高级搜索系统**：现代化的浮动搜索面板，支持逻辑过滤。
- **原生级 UI 体验**：完美适配 Material Design，支持自动暗色模式。
- **智能分类**：自动隔离未分配文档。

### 🚀 快速开始

#### 官方下载

建议通过 Chrome 网上应用店安装以获得最佳体验：
[**Chrome Web Store 下载地址**](https://chromewebstore.google.com/detail/notebooklm-enhancer/geedlihiipifeifhnmjhmnmkffanlhpg)

#### 开发者模式安装

1. 克隆此仓库：
   ```bash
   git clone https://github.com/MinghuaLiu1977/notebooklm-ext.git
   ```
2. 打开 Chrome 扩展程序页面 (`chrome://extensions/`)。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择 `extension` 目录。

## NotebookLM Enhancer (English)

NotebookLM Enhancer is an unofficial browser extension deeply customized to optimize the Google NotebookLM experience.

### 🌟 Key Features

- **Structured Directory System**: Implements folder management including creation, renaming, and drag-and-drop organization.
- **Advanced Search System**: A modern floating search panel with support for logical filtering (AND/OR).
- **Native-level UI Experience**: Adheres strictly to Material Design with automatic light/dark theme adaptation.
- **Intelligent Classification**: Automatically isolates unassigned documents into a "Pending Classification" section.

### 🚀 Quick Start

#### Official Installation (Recommended)

Install via the Chrome Web Store for the best experience and automatic updates:
[**Download from Chrome Web Store**](https://chromewebstore.google.com/detail/notebooklm-enhancer/geedlihiipifeifhnmjhmnmkffanlhpg)

#### Developer Mode Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/MinghuaLiu1977/notebooklm-ext.git
   ```
2. Open the Chrome Extensions page (`chrome://extensions/`).
3. Enable **"Developer mode"**.
4. Click **"Load unpacked"** and select the `extension` directory.
